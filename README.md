This is a starter template for building web3 projects using my preferred stack:

- Next.js
- Styled Components
- Styled JSX (with Sass support)
- Viem
- Wagmi
- RainbowKit

Run the project with `yarn dev` and visit `http://localhost:3000` to see the result.
